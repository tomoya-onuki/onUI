onUI

The onUI is a library that allows you to easily install UI components by simply adding classes to html.

Instllation

Usage

Just load the css and js.
<head>
    <link rel="stylesheet" href="onui.css">
    <script src="onui.js"></script>
</head>

Toggle Switch

<input type="checkbox" id="toggle" class="onui-toggle-switch">
<label for="toggle">Toggle</label>

Toggle Button

Toggle Button
<div class="onui-toggle-btn">Toggle Button</div>

Multi Slider

<input type="range" id="slider" class="onui-multi-slider">

Font Selector

<div id="font-selector" class="onui-font-selector">
    <option name="sans-serif"></option>
    <option name="serif"></option>
    <option name="Arial"></option>
    <option name="Times New Roman"></option>
</div>

Accordion Menu

<div id="accordion" class="onui-accordion">

    <option name="Item2">sample2 sample2 sample2 sample2 sample2</option>
</div>
