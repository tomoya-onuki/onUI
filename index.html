onUI

The onUI is a library that allows you to easily design UI components by simply adding classes to html. You don't need to write javascript code.

Quick Start

Download

  • onui.(6KB)
  • onui.js (16KB)

Usage

Just load the and JS, and write the <onui> element.
<head>
    <link rel="stylesheet" href="onui.css">
    <script src="onui.js"></script>
</head>

<body>
    <onui></onui>
</body>

Toggle Switch

<onui type="toggle-switch" id="toggle"></onui>
<label for="toggle">Toggle Switch</label>
copied! content_copy
  • Type is toggle-switch.
  • You can also use the <label> element.

Style

  • Add the "cheked" attribute when initializing in the selected state.
  • Available properties for specifying style are "background-color", "color", "border-color", "border-radius", "border-style", "border-width", "accent-color", and "margin".
    • "background-color" is the color of the background of the toggle switch when it is OFF.
    • "border-color" is the color of the toggle switch outline.
    • "color" is the color of the toggle switch knob.
    • "border-radius" is radius of corners. When property is "border-radius: r0 r1", r0 is corners of outline and r1 is corners of knob.
    • "accent-color" is the color of the background of the toggle switch when it is ON.

Style Attribute

<onui type="toggle-switch" checked
                    style="background-color:bisque;
                            border: black 1px solid;
                            border-radius: 2px 2px;
                            color: black;
                            accent-color: orange;"></onui>

Multi Slider

<onui type="range"></onui>
copied! content_copy
  • Type is range.

Style

  • "min", "max", and "step" can be specified in the same way as the HTML standard <input type="range"> element.
  • Available properties for specifying style are "background-color", "color","width", "height", and "margin".
    • "value" is specified as "v0,v1". v0 is the smaller value and v1 is the larger value.
    • "background-color" is the color of the background of the slider.
    • "border-color" is the color of the knob outline.
    • "color" is the color of the knob.
    • "accent-color" is background color between knobs.

Style Attribute

<onui type="range" value="2,8" min="0" max="10" step="1" 
                    style="background-color:black;
                            border-color: black; 
                            color: white; 
                            accent-color: crimson;
                            width: 300px; 
                            height: 5px;"></onui>
copied! content_copy

<style>

        background-color: black;
        border-color: black;
        color: white;
        accent-color: crimson;

Font Selector

<onui type="font">
    <option name="sans-serif"></option>
    <option name="serif"></option>
    <option name="Arial"></option>
    <option name="Times New Roman"></option>
</onui>
copied! content_copy
  • Type is font.
  • Choices must be written in <option> elements and added as children of <onui> elements.

Style

  • Available properties for specifying style are "background-color", "color", "border-color", "border-radius", "border-style", "border-width", "accent-color", "text-align", "width", "height" and "margin".
    • Add the "cheked" attribute when initializing in the selected state.
    • "background-color" is the color of the background.
    • "color" is the color of the text.
    • "border-color" is the color of the outline.
    • "border-style" is the style of the outline.
    • "border-radius" is rounds the corners of the outline.
    • "border-width" is width of the outline.
    • "accent-color" is the color of the highlight when hovering the mouse over a choice.

Style Attribute

<onui type="font" style="background-color:gray; 
                            color: #eee;
                            border: dimgray ridge 3px;
                            border-radius: 0px;
                            accent-color: crimson;
                            width: 200px;
                            height: 20px;">
        <option name="sans-serif"></option>
        <option name="serif"></option>
        <option name="Arial"></option>
        <option name="Times New Roman"></option>
</onui>
